# Projet 2 - Riding Cities

## Contexte

Ce projet a été réalisé dans le cadre du **Projet 2 de la formation Développeur IA d'OpenClassrooms**.

L'exercice repose sur une mise en situation professionnelle au sein de l'agence web fictive **Webonzaï**. En tant que développeur junior stagiaire, la mission consiste à mettre à jour la page web de **Riding Cities**, une association sportive promouvant la pratique du skate en région.

## Objectifs pédagogiques

Ce projet a pour objectif de découvrir et mettre en pratique les fondamentaux du développement web en utilisant :

* HTML5
* CSS3
* La structuration d'une page web
* La mise en forme et le stylage d'un site
* L'intégration d'une maquette graphique
* Les bonnes pratiques de développement web

## Mission

À partir des ressources fournies (site existant, code source et maquette graphique), il fallait :

* Analyser les besoins du client ;
* Comprendre la structure existante du site ;
* Modifier et enrichir le contenu de la page ;
* Appliquer les styles nécessaires pour respecter la maquette ;
* Utiliser correctement les balises HTML5 et les propriétés CSS.

## Compétences développées

* Création et structuration de pages web avec HTML5 ;
* Mise en forme avec CSS3 ;
* Utilisation d'un éditeur de code (Visual Studio Code) ;
* Gestion de projet avec Git et GitHub ;
* Lecture et compréhension d'un cahier des charges.

## Technologies utilisées

* HTML5
* CSS3
* Git
* GitHub
* Visual Studio Code
